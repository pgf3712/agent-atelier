# Changelog

All notable changes to Paula & Indy's Agent Atelier are documented here. The project follows Keep a Changelog and semantic versioning.

## [0.1.0] - 2026-07-18

### Added

- Eleven bilingual learning chapters with distinct interactive labs.
- Deterministic provider, bounded agent loop, typed tool contracts and evidence validation.
- State/memory separation, explicit planning and safe public observability.
- Untrusted-content guardrails, scoped human approval and abstract cost/latency accounting.
- Integrated Research Brief application and five-scenario deterministic evaluation.
- Cooperative wall-clock budget with deterministic clock tests.
- Local-only preview boundary and restrictive browser security headers.
- Eleven independent transparent Paula and Indy chapter portraits, a sleeping sidebar mascot and a progressive pixel-art studio.
- Study, Learning and Workshop modes with guided predictions, code choices, contextual hints and reasoned feedback.
- Paula's chapter-specific technical notebook and Indy's progressive optional help.
- Optional original procedural cozy-jazz audio and an animated completion reward.
- Windows preview and verification launchers.
- GitHub Actions quality workflow, public-release audit and bilingual interview guide.
- Copyright notice, project asset credits and GitHub contribution templates.

### Security

- Non-loopback preview binding now requires explicit `--allow-remote` opt-in.
- Provider proposals remain separated from application-owned tool execution, budgets and citation validation.

### Known limitations

- The default provider is deterministic and the optional real provider is deliberately limited to the integrated final chapter.
- There is no authentication, durable database storage, live web search or hosted deployment.
