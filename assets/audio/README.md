# Audio Assets

The current version includes `agent-atelier-ambient.wav`, an original 40-second cozy-jazz background loop generated reproducibly by `design/generate_audio.py`. It uses a slow Cmaj7–Am7–Dm7–G9 turnaround and a soft sample-free electric-piano timbre. The web interface starts it only after an explicit learner action, provides volume and mute controls, and displays playback status.

Reserved categories:

- `music/`: future alternative background themes, always off by default.
- `ambient/`: rain, room and subtle environmental loops.
- `ui/`: short interaction and status cues.
- `cat/`: rare, subtle Indy sounds.

Only original or clearly reusable audio may be added. Prefer CC0. Every external file requires a credit record, exact license text/link, source, creator, modification notes and a checksum. “Royalty-free” alone is not sufficient.
