# Code of Conduct

Agent Atelier is an educational project. Participation should make learning safer, clearer and more welcoming.

## Expected behaviour

- Discuss ideas and code with respect.
- Give specific, constructive feedback.
- Welcome questions from people with different levels of experience.
- Respect privacy, attribution and licensing boundaries.
- Report security issues privately rather than publishing secrets or exploitable details.

## Unacceptable behaviour

Harassment, discrimination, personal attacks, sexualized attention, deliberate intimidation, doxxing and publication of another person's private information are not accepted.

## Enforcement

Project maintainers may edit or remove contributions and restrict participation when behaviour conflicts with these expectations. Reports should include only the information needed to understand the incident. A private contact channel will be added before the first public release.

This document applies to project spaces and to public interactions where someone represents Agent Atelier.
